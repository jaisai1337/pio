# PIO INSTALLTION
## Download & Installing Ubuntu in Termux App 
```
wget https://raw.githubusercontent.com/jaisai1337/pio/main/pio_01.sh
chmod +x pio_01.sh
./pio_01.sh
```
### Login into Ubuntu
```
proot-distro login ubuntu
```
